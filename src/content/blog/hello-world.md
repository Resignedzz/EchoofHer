## 你好 👋

这是我的第一篇博客文章。如果你能看到这篇内容，说明博客系统已经正常运行了。

### 这个博客是用什么搭建的？

- [Astro](https://astro.build) — 现代静态站点生成器
- [Tailwind CSS](https://tailwindcss.com) — 原子化 CSS 框架
- [Pagefind](https://pagefind.app) — 静态全文搜索引擎
- [Giscus](https://giscus.app) — 基于 GitHub Discussions 的评论系统

### Markdown 支持

支持标准 Markdown 和一些扩展语法：

**粗体**、*斜体*、~~删除线~~

> 引用文本，适合用来标注重要信息。

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

### 代码高亮

Astro 内置了 Shiki 代码高亮，支持多种语言：

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```
